# PDFMaster - Free Online PDF Tools

🌐 **Live Demo & Full Suite: https://pdfeasz.com**

Free, privacy-first PDF tools for everyone. Merge, split, compress, and convert PDFs 100% locally in your browser. No uploads, no registration, no watermarks.

---

## ✨ Tools Included
- **PDF Merger**: Combine multiple PDFs into one file.
- **PDF Splitter**: Extract specific pages or split by range.
- **PDF Compressor**: Reduce file size without losing quality.
- **Image to PDF**: Convert images to standard PDF documents.

## 🚀 Key Features
✅ **100% Free**: No hidden fees or premium tiers.
✅ **Privacy First**: All processing happens locally. Your files never leave your device.
✅ **No Registration**: No sign-ups, no emails required.
✅ **No Watermarks**: Clean, unbranded output every time.
✅ **Cross-Platform**: Works on Windows, Mac, iOS, and Android.

## 🛠️ How to Use Locally
This project is open-source and can be run directly in your browser:
1.  Clone this repository.
2.  Open `index.html` in any modern web browser.
3.  Use the tools offline, just like on the website.
